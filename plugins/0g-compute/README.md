# 0G Compute Plugin

Decentralized AI inference and fine-tuning platform for Claude Code.

## Overview

This plugin provides comprehensive support for the 0G Compute platform, enabling Claude to help you:

- Build AI inference applications (chatbots, image generation, speech-to-text)
- Fine-tune models on distributed GPUs
- Manage accounts and billing
- Integrate the 0G Compute SDK correctly

## Skills Included

This plugin provides skills that activate automatically when you mention relevant topics:

- AI inference patterns
- Model fine-tuning workflows
- Account management operations
- SDK integration best practices

## Activation

The skill activates automatically when you mention:
- "0g compute"
- "inference"
- "processResponse"
- "/0g-compute" (explicit command)
